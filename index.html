Editing

i/I - insert before cursor or beginning of line
a/A - append after cursor or end of line
o/O - enter insert mode but insert line after current one or before current line
R - enter replace mode
s - delete character under cursor and go into insert mode
x - delete character under cursor
[#]c - change a character or change three characters 3c
cc/S/C - change/replace whole line cc/S or change/replace to end of lineC
ciw - change word under cursor
caw - change around word (including trailing space)
cw - change word under cursor
dw/d#w - delete word or delete # of words
diw - Delete inside word
daw - Delete word and trailing space
di"/di'/di)/di} - Delete inside double quotes, single quotes, parenthese, or curly braces
dd/#dd - delete a line (and copy it to system register), delete # lines
d$/D - delete from cursor up to end of line
:5,25d - delete lines 5-25
:.m 12 - move current line after line 12
:5,7m 21 - move lines 5-7 after line 21
[#]r[char] - replace number of characters [#] with character [char]. 3rf
ddp/ddkP - move line down or moe line up
u - undo last change
:undolist - list your possible undo changes using u
J - join current and below line with space in between
gJ - join current and below line with no space between

Visual Mode

v - enter visual mode
V - enter visual mode highlighting the current line
viw - visually select word without spaces
vaw - visually select work with trailing space
vi[ - visully select inside next occurence of [
gv - reselect the last visual selection

Moving Visually Selected Text

:le 10 - takes the visually selected text and adds 10 spaces to the left of this selected text starting from beginning of line

Moving Around

h -
j -
k -
l -
H/M/L - Move to top H, middle M or bottom of screen L
w/W - jump ahead to start of next word or space-separated word
e/E - jump ahead to end of next word or space-separated word
b/B - jump behind to start of previous word
gg/G/#G - Jump to 1st line of file, last line of file, or #th-line of file
#% - Move to the # % of file. 25% would move to the 25% percent of file
#| - Move to # column in line
[Enter]/+ - Move to the next non-blank character on line
-
:# - move to line # where # is a number.
/[chars] - search for a string below cursor
?[chars] - search for a string above cursor
n/N - search for next or previous search occurence using / or ?. Or keep hitting / or ? and [Enter] will do same thing
( ) { } - move to beginning or end of sentence or beginning/end of paragraph (or next blank line)
0/^/$i/g_ - go to beginning of line, start of first non-blank character in line, end of line, or last non-blank character in line
fz/Fz/tz/Tz - jump to next f or previous F character 'z', jump to next t or previous T character 'z' but behind one character
,/; - using f/t commands above, repeat in the command direction or previous direction
zt/zz/zb - orient screen so line is at (top|center|bottom) of screen
-

Repeating Commands

. - Repeat last command performed
#{command} - Repeat command # times
-
-

Copy, Cut and Paste / Registers

yy/Y - Copy/Yank current line
2yy - Copy/Yank current line and line below it
y$ - Copy/yank from current cursor to end of line
p/P - paste before cursor and after cursor
"ap - paste from register a
"+p - paste from the system clipboard
".p - paste from last inserted text
"0-9p - paste from delete history
:reg[isters] - list registers and their contents
]p - paste line with right indent

Split Windows

:sp[lit] [file] - split window horizontally opening [file]
:vs[plit] [file] - split window vertically opening [file]
Ctrl+ww - switch to other split window
Ctrl+w[dir] - move to split window using direction ([dir] can be hjkl or arrow keys)

Folds

zf - create fold at cursor position
zd - delete fold under cursor
za - toggle fold at cursor location
